Module 2: Computation Tools

Introduction to Hydrographic Surveying Computation Tools Nautical Science Environment Science Positioning Underwater Acoustics Water Levels QA/QC Remote Sensing Hydrographic Practice Hydrographic Data Management Comprehensive Final Field Project

Hydrographic surveying is a science! On Module 2, you will be exposed to math and programming concepts in application to hydrographic survey needs. You will learn to build simple processing tools in Python for file conversion and scripting, applied to hydrographic data handling. Together with math and programming, you will also learn about statistics, theory of errors and other related topics.
